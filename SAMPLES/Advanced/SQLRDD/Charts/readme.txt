--------------------------------------
 Requirement:
--------------------------------------
- installed MySQL server version 8.0 with the samples

For additional downloads and the source of MySQL, visit
  http://dev.mysql.com/downloads/

- installed MySQL Connector/ODBC (formerly MyODBC), the driver that
enables ODBC applications to communicate with MySQL servers.

For more information on MySQL Connector/ODBC visit
  http://dev.mysql.com/doc/connector-odbc/en
